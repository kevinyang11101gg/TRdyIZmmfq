# Python设备仓储入库管理系统

## 核心功能
- 登录注册
- 库存信息可视化大屏
- 仓库信息管理
- 物料信息管理
- 员工管理
- 入库单管理
- 出库单管理
- 供应商管理
- 客户信息管理
- 绩效考核管理
- 待上架任务管理
- 个人中心

## 适用场景
- 企业内部设备仓储管理
- 设备出入库流程监控
- 库存信息实时查看与分析
- 员工绩效评估
- 供应商与客户关系管理

## 技术特点
- 使用Django框架进行后端开发
- MySQL数据库进行数据存储
- 前端采用Vue.js框架
- 系统界面友好，易于操作
